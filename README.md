   內容含：
   - settings payload / global settings / scenario runtime
   - edited 與 normalized 三組組合
   - single-path 年度路徑
   - validation checks
   - diagnostic summary
   - light log
   - 若已有 Monte Carlo summary/detail，也一起打包

2. 產生完整 LOG / AI 驗證包（較慢）
   - 會額外產生 Monte Carlo path replay
   - 可下載完整 LOG JSON
   - 可下載完整 AI 驗證包 ZIP

目的：
讓使用者可以把整包資料丟給 AI 或人工，驗證整體計算，不必只靠截圖或手動逐項下載。
